Tarea 4 Prog 1

Este es un programa basico de registro de clientes, cuenta con funciones como, agregar, modificar y eliminar datos.

La carpeta Interfaces cuenta con 6 archivos los cuales son: clientes, eliminar, login, modificar, nuevo y registro.

Cliente: en este mostramos la visualizacion de la tabla, como tambien los botones nuevo, modiicar, eliminar y cerrar sesion.
Eliminar: en este podremos eliminar un registro, solamente con ingresar un usuario(registrado).
Login: en este ingresaremos al sistema(Cliente) y contiene botones para registrarse y logearse.
Modificar: en este podremos modificar datos de algun usuario registrado, tendremos un boton "Buscar" el cual 
le añadiremos el usuario a modificar y te traeria todos los datos de ese usuario.
Nuevo: en este este podremos registrar un nuevo usuario.
Registro: en este, es donde nos registraremos para luego ingresar al sistema(Cliente).


La carpeta Conexion cuenta con 3 archivos los cuales son: DBCusuario, DBconexion, DBregistro.

DBCusuario: aqui encontramos el metodo para validar los usuarios y el email, para que no haya duplicados y el email este correctamente
y el metodo de confirmar contraseña.
DBconexion: aqui es donde se realiza la coneccion para llamar la base de datos a nuestro programa.
DBregistro: aqui tenemos Strings almacenadas para usalas en el registro.


Extras:

*Si dejamos los campos vacios en el registro nos sale una alerta.
*Si las contraseñas no coinciden en el registro sale una alerta.
*Si intentamos iniciar sesion sin datos sale una alerta.
*Una vez agregamos, modificamos o aliminamos algun usuario, este se vera reflejado automaticamente en la pantalla principal



Este proyecto fue realizado con eclipse y mysql
